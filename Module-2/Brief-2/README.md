# Présentation des choix éthiques effectués

## Types de variables et les risques associés

`nom, prenom`

Identification directe interdite par le RGPD (PII: Personally Identifiable Information)

`sexe`

Éviter les biais de genre dans l'octroi de crédits

`nationalité_francaise`

Risque de discrimination selon l'origine

`niveau_etude`

Aucun risque

`region`

Peut etre utilisé comme proxy pour determiner une origine social

`age`

Aucun risque éthique identifié / ou peut être discriminant

`revenu`

Aucun risque éthique identifié

`loyer`

Aucun risque éthique identifié
